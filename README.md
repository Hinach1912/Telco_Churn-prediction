# Telco_Churn-prediction
This project implements an end-to-end machine learning pipeline for predicting telecom customer churn using the IBM Telco Customer Churn dataset. The notebook demonstrates preprocessing, model training, hyperparameter tuning, evaluation, and model serialization for deployment.
# Project Features
• End-to-end scikit-learn Pipeline
• ColumnTransformer preprocessing
• Logistic Regression and Random Forest models
• Hyperparameter tuning using GridSearchCV
• Model evaluation
• Save trained pipeline using joblib
# Dataset
IBM Telco Customer Churn dataset containing customer demographics, services, billing information, and churn labels.
Workflow
1. Install libraries
2. Load dataset
3. Exploratory Data Analysis
4. Data preprocessing
5. Build ML pipeline
6. Train Logistic Regression and Random Forest
7. Hyperparameter tuning with GridSearchCV
8. Evaluate performance
9. Save best model with joblib
# Libraries
pandas, numpy, scikit-learn, matplotlib, seaborn, joblib
Model Evaluation
Performance is evaluated using standard classification metrics such as Accuracy, Precision, Recall, F1-score, Confusion Matrix, and Cross-validation.
Project Structure
Notebook contains data loading, preprocessing, pipeline construction, model training, tuning, evaluation, and model export.
